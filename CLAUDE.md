# GNJ Média — Site corpo (gnjmedia.ca)

## Stack
- Vanilla HTML/CSS/JS, un seul index.html, tout inline. ZÉRO framework, zéro build.
- Déployé sur Cloudflare Pages (racine = output, pas de build command).
- Mobile-first, accessible, léger.

## Médias
- Vidéos hero : assets/video/rockfish.mp4, shineup.mp4, jardin.mp4 (web-optimisés, muets, autoplay loop).
- Masters .MOV : HORS du repo (trop lourds pour git + Cloudflare).

## Identité de marque
- Base near-black #0A0A0B, texte blanc. Section équipe sur crème #F4F3EF, texte noir.
- Accents : bleu Meta #1877F2 (mark glow section Process), or #FFB200 (avis Google).
- Typo : Clash Display (titres), Inter (corps), JetBrains Mono (micro-labels, dont verticaux).
- Texture : grille blueprint subtile (84px). PAS de film grain.
- Logomark : SVG 5 barres, réutilisé nav + ghosté géant en background.
- Ton : premium, direct, confiant. Français québécois partout.

## Positionnement
Studio vidéo & publicité Meta au Québec. « On transforme les vues en clients. »
Contenu organique (Reels) → créatives pub Meta → leads. Cible : PME + B2B Québec.

## Sections (ordre)
Hero → Process → Résultats → Avis (Google 5/5) → Équipe → Booking → Footer.
